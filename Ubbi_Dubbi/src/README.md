# Ubbi Dubbi Translator

A simple Python program that translates English words and sentences into **Ubbi Dubbi ** .

## What is Ubbi Dubbi?

Ubbi Dubbi is a language game in which **"ub"** is added before every vowel sound.

**Rule:**

* Every vowel (`a`, `e`, `i`, `o`, `u`) becomes `ub` + vowel.
* Consonants remain unchanged.

**Examples:**

* `cat` → `cubat`
* `hello` → `hubellubo`
* `apple` → `ubapplube`

## Features

* Translate single words
* Translate complete sentences
* Interactive mode for continuous translation
* Exit using one of the following commands:

  * `exit`
  * `quit`
  * `bye`
  * Press **Enter** on an empty line

## Requirements

* Python +3.7

## How to Run

```bash
python main.py
```
